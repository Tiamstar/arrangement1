## 寒招打卡流程  
大家详细阅读pdf要求  
线下宣讲打卡必须当日完成所有审批流程（即队长审批→分团长审批→队员销假）
遇到不合适的打卡，大家驳回的时候记得填写理由（线上不打卡、队友重复提交审批等）
两支队伍一同宣讲的学校，分队伍打卡，方便后期核算

这段时间，只有完成打卡全流程，显示为【已通过】的宣讲才可以记录线下分数
